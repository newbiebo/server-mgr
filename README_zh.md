# server-mgr

[README](README.md) | [中文文档](README_zh.md)

server-mgr 是一个家庭私人服务器管理应用，包括灵活的资源备份策略，苹果bark应用消息通知，邮件通知，服务器监控等。

## 为什么使用server-mgr？

在家庭服务器中部署server-mgr，server-mgr可以帮你：

* 省去购买昂贵的磁盘整列。
* 可用极少的资源保证家庭数据安全。
* bark应用消息、邮件实时推送服务器状态。
* 服务器状态监控
* 提供服务器管理页面
